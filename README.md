# Rock Paper Scissors 🎮

Simple console-based Rock Paper Scissors game written in C++.

## Description
Play against the computer with random moves.  
The program shows who wins each round and keeps the game interactive.

## How to Play
1. Run the program in any C++ compiler or Visual Studio.
2. Enter your choice:
   - Rock
   - Paper
   - Scissors
3. The computer will make a random choice.
4. The winner of the round will be displayed.

## Language
C++
